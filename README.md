# 業務効率化・データ分析システム (DX：PHP, SQLite, Power BI)
  
## 概要
  架空の社内タスクデータを対象に、「データの入力（PHP：Webアプリ）→蓄積（DB）→可視化（BI）」までのデータパイプラインを一気通貫で構築しました。
  
## 使用技術
- PHP
- SQLite
- Power BI

## 実装内容
1. Frontend (PHP):  
   - 日報入力用Webフォームの作成。
   - データベース(SQLite)上のマスタデータと連携した動的なプルダウン生成、入力バリデーションの実装。
2. Database (SQLite):  
   - 業務ログデータの設計・構築。
   - SQLを用いたデータの整合性検証。
3. BI (Power BI):  
   - 分析しやすいデータ構造（スタースキーマ）を設計し、部署や担当者別でのドリルダウン分析を実現。
   - DAX関数を活用し、「実稼働時間」や「未完了タスク数」などの重要指標（KPI）を動的に算出・可視化。
