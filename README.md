# 業務効率化・データ分析システム (DX：PHP, SQLite, Power BI)
  
## 概要
  架空の社内タスクデータを対象に、「データの入力（Webアプリ）→蓄積（DB）→可視化（BI）」までのデータパイプラインを一気通貫で構築しました。
  
## 使用技術
- PHP
- SQLite
- Power BI

## 実装内容
1. Frontend (PHP):
   日報入力用Webフォームの作成。
   データベース(SQLite)上のマスタデータと連携した動的なプルダウン生成、入力バリデーションの実装。
3. Database (SQLite): 業務ログデータの設計・構築。SQLを用いたデータの整合性検証。
4. BI (Power BI): 工数管理ダッシュボードの設計。
                  スタースキーマ（リレーション）構築およびDAX関数を用いたKPI（未完了タスク数、稼働時間）の算出。
